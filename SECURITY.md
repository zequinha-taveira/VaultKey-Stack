# Security Policy

## Supported Versions

Only the latest version of VaultKey Stack is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | ✅ (Scheduled)     |

## Reporting a Vulnerability

We take the security of VaultKey Stack very seriously. If you believe you have found a security vulnerability, please do **not** open a public issue. Instead, send an email to security@example.com (Placeholder).

Please include:
- A description of the issue.
- Steps to reproduce.
- Any potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary.
